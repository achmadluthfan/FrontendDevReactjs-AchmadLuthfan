# Project Setup

## Requirements

- **React.js Version**: 18.3.1
- **Node.js Version**: v20.10.0
- **Package Manager**: npm

## Installation

1. Clone repository:
   ```bash
   git clone https://github.com/achmadluthfan/sekawan-media.git
   cd sekawan-media
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

## Running the Project

- Menggunakan npm:
  ```bash
  npm run dev
  ```
